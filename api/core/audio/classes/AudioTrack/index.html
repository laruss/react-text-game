Class: AudioTrack

Defined in: audioTrack.ts:35

Individual audio track with reactive state management.

AudioTrack wraps the HTMLAudioElement API with Valtio reactive state, making it easy to integrate with React components and the game's save system.

Features:

  • Reactive state updates via Valtio
  • Automatic persistence with save/load
  • Volume, loop, and playback rate controls
  • Fade in/out effects
  • Event-driven state synchronization

Example​

const music = new AudioTrack('assets/music.mp3', {
  id: 'bg-music',
  loop: true,
  volume: 0.7,
});

await music.play();
music.setVolume(0.5);
