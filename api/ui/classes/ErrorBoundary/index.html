Class: ErrorBoundary

Defined in: packages/ui/src/components/ErrorBoundary/ErrorBoundary.tsx:17

ErrorBoundary catches all JavaScript errors in the application:

  • Rendering errors
  • Event handler errors
  • Async errors (promises, setTimeout)
  • Global uncaught errors

contextType?​

static optional contextType: Context<any>

Defined in: node_modules/.bun/@types+react@19.2.2/node_modules/@types/react/index.d.ts:922

If set, this.context will be set at runtime to the current value of the given Context.

Example​

type MyContext = number
const Ctx = React.createContext<MyContext>(0)

class Foo extends React.Component {
  static contextType = Ctx
  context!: React.ContextType<typeof Ctx>
  render () {
    return <>My context's value: {this.context}</>;
  }
}

See​

https://react.dev/reference/react/Component#static-contexttype
